# 🌍 GeoIntel - OSINT Geolocation Intelligence

A browser-based forensic geolocation tool that analyzes images to determine where they were taken using AI-powered detection of infrastructure, architecture, vegetation, and environmental clues.

## ✨ Features

- **AI Image Analysis** — Detects brick patterns, utility markings, street signs, license plates, vegetation, shadows, and architectural styles
- **Confidence Scoring** — Transparent 0-100% confidence with evidence breakdown
- **Interactive 3D Globe** — Three.js visualization with orbiting satellite animation during analysis
- **Split-Screen Comparison** — User photo vs street/satellite imagery
- **Evidence Overlay** — Highlights exactly which pixels contributed to location decision
- **Multiple Map Layers** — Satellite, street, terrain, and infrared vegetation views
- **PDF Export** — Professional forensic reports
